## Setup
To run this project install it locally:

```bash
git clone https://github.com/sa-keshav/AdventOfCode2020.git
```
then navigate to each project and run the code.
<br>
<br>
For example to run the first code do so:
```bash
cd /1
./adventOfCode1.py
```
